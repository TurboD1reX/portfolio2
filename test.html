Project Description

A simple web application to display a list of bus lines in Gliwice, Poland. The application uses the OpenData API to fetch and display the bus lines. My friend was interest in making this web application because he had the notes about buses. He had their serial number, does the bus have wifi, does the bus have 4-chairs seats.
